Overview

This is the user interface (UI) style guide for The Service Innovation Lab

The site uses a custom set of styles that inherit from the SIL brand guidelines to create style that is professional, unique, and informative. The guide has a two-fold purpose:

  1. Speed up onboarding of future maintainers of the site. While most of this guide addresses content needs, there is some technical documentation for how we have extended Jekyllâ€™s out-of-the-box functionality.
  2. Serve as a living reference for SIL brand of styles.

If you are updating or adding a new blog post, go to Layouts to get started, or go directly to Blog components to learn more about enriching your blog post.

Adding a project page

If you are updating or adding a new project page, go to Layouts to get started.

Overriding the Design System

While te SIL relies heavily on the NZ Government designs/styles, there are several instances where we have overridden styles with our own set of custom stylesheets. This is primarily to align ourselves more directly with the SIL brand set of fonts, colors, and logos. Additional overrides were made to create custom layouts or components to address user needs that are specific to the marketing of the SIL brand.
